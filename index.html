GitHub Language Popularity

In 1843, mathematician Ada Lovelace published an algorithm to calculate a sequence of Bernoulli numbers. This algorithm is considered to be the first computer program ever developed. Since then, numerous programming languages have been developed for a programmer to choose from. However some are more popular then others. You can use this app to evaluate the popularity of several languages being used today via current data from GitHub. The results are based on the mean amount of stars received by the top 30 most starred repositories for the selected language. Please choose a language below to get started.
